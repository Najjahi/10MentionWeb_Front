rowspan / colspan

" roxspan"" est un attribut qui permet d'indiquer le nombre de lignes qu'une cellule <td> de tableu <table> peut couvrir. Cet attribut est utile si vous souhaitez qu'une cellule s'etande verticalement sur plusieurs lignes. Exemple si on met rowspan="5" alors notre cellule occupera l'espace de 5 cellules verticalement. Cependant il est important de noter que lorsqu'on utilise cette manipulation il faut retitrer les correspendants des autres lignes

"colspan" est un attribut qui permet d'indiquer le nombre de colonnes qu’une cellule <td> de tableau <table> doit couvrir. Cet attribut est utile lorsque vous souhaitez qu’une cellule s’étende horizontalement sur plusieurs colonnes .Exemple si on met colspan="3", alors notre cellule occupera l'espace de 3 cellules horizontalement.
