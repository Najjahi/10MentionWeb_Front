C -Direction

flexbox permet d'agencer ces éléments dans le sens que l'on veut. avec flex-direction on peut les positionner verticalement ou encore les inverser . il peut prendre les valeurs suivants :
- row : organisés sur une ligne (par defaut)
- column : organisés sur une colonne
- row-reverse : organisés sur une ligne mais en ordre inversé
- column-reverse : organisés sur une colonne mais en ordre inversé
